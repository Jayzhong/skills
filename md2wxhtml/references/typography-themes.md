# Typography Themes Reference

This document describes all available typography themes for the MD2WxHTML converter.

## Theme List

### Classic

**Style:** Traditional serif with generous spacing

- **Font:** Georgia, "Noto Serif SC" serif
- **Heading:** Bold with border-bottom on h1
- **Blockquote:** Warm brown border (#8b7355), cream background
- **Best for:** Traditional articles, essays, literary content

### Modern

**Style:** Clean sans-serif with tight spacing

- **Font:** -apple-system, sans-serif
- **Heading:** Semi-bold (600), subtle border on h2
- **Blockquote:** Gray border (#dfe2e5), no background
- **Best for:** Tech articles, modern content, documentation

### Minimal

**Style:** Ultra-clean with maximum whitespace

- **Font:** system-ui, sans-serif
- **Heading:** Light weight, generous margins
- **Blockquote:** Thin 2px border, minimal styling
- **Best for:** Focus on content, distraction-free reading

### Warm

**Style:** Warm earth tones with soft backgrounds

- **Font:** Georgia, serif
- **Heading:** Warm brown colors (#3d3426)
- **Blockquote:** Golden border (#d4a574), cream background
- **Best for:** Lifestyle content, personal blogs, warm topics

### Serif

**Style:** Book-like typography with centered h1

- **Font:** "Noto Serif SC", serif
- **Heading:** Centered h1, strong hierarchy
- **Paragraph:** 2em text indent
- **Best for:** Long-form articles, book-style content

### Book

**Style:** Chapter-style with literary aesthetic

- **Font:** Charter-like serif
- **Heading:** Small-caps h1, chapter numbering style
- **Blockquote:** Wide margins, no border
- **Best for:** Novels, literary articles, serialized content

### Tech

**Style:** Code-like aesthetic with monospace accents

- **Font:** SF Pro Text, sans-serif
- **Heading:** Monospace font family
- **Code:** Dark background (#1e1e2e), blue accents
- **Best for:** Programming tutorials, technical documentation

### Dark

**Style:** Dark mode with light text

- **Font:** -apple-system, sans-serif
- **Colors:** Light text (#e6e6e6) on dark (#1a1a1a)
- **Links:** Blue accent (#58a6ff)
- **Best for:** Night reading, developer content

## Font Size Presets

|--------|------|----|----|----|----|
| S | 14px | 24px | 20px | 17px | 13px |
| M | 15px | 26px | 22px | 18px | 14px |

## Line Height Presets

| Preset | Value | Description |
|--------|-------|-------------|
| Compact | 1.6 | Denser text, more content visible |
| Standard | 1.8 | Balanced readability |
| Relaxed | 2.0 | Maximum breathing room |

## Border Radius Presets

| Preset | Value | Applied To |
|--------|-------|------------|
| Off | 0 | No rounded corners |
| Low | 4px | Subtle rounding |
| Medium | 8px | More pronounced rounding |
