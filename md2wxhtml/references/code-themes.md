# Code Highlight Themes Reference

This document describes all available code syntax highlighting themes.

## Theme List

### GitHub Light

- **Background:** #ffffff (white)
- **Foreground:** #24292e (dark gray)
- **Style:** GitHub's default light theme
- **Best for:** Light mode articles, documentation

### GitHub Dark

- **Background:** #0d1117 (near black)
- **Foreground:** #c9d1d9 (light gray)
- **Style:** GitHub's dark mode
- **Best for:** Dark mode articles, developer content

### Monokai

- **Background:** #272822 (dark green-gray)
- **Foreground:** #f8f8f2 (off-white)
- **Style:** Classic Sublime Text theme
- **Best for:** Colorful syntax highlighting, vibrant code

### Dracula

- **Background:** #282a36 (dark purple-gray)
- **Foreground:** #f8f8f2 (off-white)
- **Style:** Popular dark theme with vibrant colors
- **Best for:** Purple-accent designs, modern dark mode

### Solarized Light

- **Background:** #fdf6e3 (cream)
- **Foreground:** #657b83 (gray)
- **Style:** Ethan Schoonover's precision light palette
- **Best for:** Reduced eye strain, warm light theme

### Solarized Dark

- **Background:** #002b36 (dark teal)
- **Foreground:** #839496 (gray)
- **Style:** Ethan Schoonover's precision dark palette
- **Best for:** Extended coding sessions, professional dark

### Nord

- **Background:** #2e3440 (polar night)
- **Foreground:** #d8dee9 (snow storm)
- **Style:** Arctic, north-bluish color palette
- **Best for:** Cool-toned designs, Nordic aesthetic

### One Dark

- **Background:** #282c34 (dark gray)
- **Foreground:** #abb2bf (light gray)
- **Style:** Atom editor's iconic dark theme
- **Best for:** Balanced colors, familiar to developers

## Supported Languages

The converter supports syntax highlighting for:

- JavaScript, TypeScript, JSX, TSX
- Python, Ruby, PHP
- Java, Kotlin, Swift
- Go, Rust, C, C++, C#
- HTML, CSS, SCSS
- JSON, YAML, XML
- Markdown, GraphQL
- Bash, Shell, PowerShell
- SQL, Dockerfile
- Vue, Svelte

Unknown languages fallback to plain text display.
